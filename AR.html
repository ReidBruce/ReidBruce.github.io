AR Tower Attack

This is my first experience using Augmented Reality technology. The game works by having the player find a flat surface to play on. Once they tap the screen, a large castle with multiple enemies will appear, and the player must clear three levels to win. The game can only be played on an Android device, but the gameplay is still really fun.

This code allowed the player to scan images to unlock weapons and power-ups. The code uses the Vuforia engine to scan the images and then spawn the objects in the game world.

This code allowed the phone to scan flat surfaces and spawn the castle. It's not very complicated code, but I find it impressive nonetheless.
